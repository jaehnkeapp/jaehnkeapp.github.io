Anleitung zum Löschen Ihres Kontos:

  1. Öffnen Sie die App auf Ihrem Gerät.
  2. Navigieren Sie zu den Einstellungen.
  3. Wählen Sie "Sicherheit" aus.
  4. Suchen Sie nach der Option "Konto löschen" und wählen Sie sie aus.
  5. Bestätigen Sie, dass Sie Ihr Konto wirklich löschen möchten.
  6. Nachdem Sie Ihr Konto gelöscht haben, werden alle Ihre Daten aus der App entfernt.
  7. Beachten Sie jedoch, dass einige Daten möglicherweise noch für einen bestimmten Zeitraum gespeichert bleiben, bevor sie endgültig gelöscht werden. Die genauen Informationen dazu finden Sie in der Datenschutzerklärung der App.
  8. Wenn Sie weitere Fragen oder Bedenken haben, können Sie sich an den Kundendienst der App wenden, um weitere Unterstützung zu erhalten.
